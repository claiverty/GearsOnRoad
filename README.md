# Gears on Road

Gears on Road é um jogo 2D desenvolvido em Python com Pygame, no estilo endless runner.

O jogador controla um carro em uma estrada de 3 pistas e deve desviar dos veículos inimigos para sobreviver o máximo possível e fazer a maior pontuação.

## Tecnologias utilizadas

- Python
- Pygame
- PyInstaller

## Funcionalidades

- Menu inicial
- Movimentação entre 3 pistas
- Carros inimigos
- Sistema de pontuação
- Recorde salvo localmente
- Explosão e som de colisão
- Tela de game over
